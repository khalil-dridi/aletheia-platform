# Aletheia Platform

## Backend User Service

### 1️⃣ Cloner projet

git clone https://github.com/khalil-dridi/aletheia-platform.git

### 2️⃣ Configurer application.properties

Remplacer CHANGE_ME par vos valeurs :

* DB password
* JWT secret
* Cloudinary keys

### 3️⃣ Lancer backend

cd user-service
mvn spring-boot:run

Backend → http://localhost:8080
Swagger → http://localhost:8080/swagger-ui.html
